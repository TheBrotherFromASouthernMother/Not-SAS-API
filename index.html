Welcome to the Semester at Sea API

The unofficial REST API of Semester at Sea

Created to provide comprehensive data on all Semester at Sea voyages since the year 2012. Simple endpoints provide access to voyage names, semesters, ports of call, academic sponsors, ships, and assoicated images.

Usage:

Get a single voyage by id

semester-at-sea.me/api/voyages/id
